# Changelog

## [Unreleased]

## [2.0.0]

### Changed
- Raised the minimum supported Julia version to 1.10.
- Migrated test infrastructure to TestItems.
- Updated SSP5 population data as provided by Benveniste et al. Re-ran all processign scripts.

### Fixed

### Added
- Added and refreshed calibration and supporting data used in recent figure reproduction workflows.

## [1.0.2] - 2024-02-08

### Changed
- Expanded compatibility support for Interpolations 0.15, XLSX 0.10, and Downloads 1.
- Improved package metadata and documentation in README updates.

### Fixed
- Applied maintenance fixes from Julia Package Butler and related project cleanup.

## [1.0.1] - 2023-06-07

### Changed
- Updated SSP585 input data to align with MimiFAIRv2 and corrected Leach 2021 input files -- see notes on rebased gases.

### Fixed

## [1.0.0] - 2022-08-25

### Added
- Initial stable release of the MimiSSPs package.
